# 工程科技问答

功能：输入工程科技问答query，返回该问题对应的answer

## 使用说明

- Method: POST
- URL: 详见系统地址
- 输入参数

| 参数 | 类型 | 是否必须 | 描述 |
| --- | --- | --- | --- |
|query|string|required|一段问答描述|

- 返回结果

| 参数 | 类型 | 是否必须 | 描述 |
| --- | --- | --- | --- |
|errno|string|是|状态码，0表示正常，-1表示服务异常|
|query|string|是|输入问答描述|
|result|string|是|答案|
|url|string|是|答案对应的网页链接|
